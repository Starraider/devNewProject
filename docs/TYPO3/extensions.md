# Common Extensions

Since some TYPO3 extensions are useful or even necessary in all DGVT website projects, it makes sense to prepare the installation and configuration already in this project template.

## List of common extensions

* [skom/dgvtbase](https://github.com/Starraider/dgvtbase)
* [bk2k/bootstrap-package](https://github.com/benjaminkott/bootstrap_package)
* [b13/container](https://github.com/b13/container)
* [helhum/dotenv-connector](https://github.com/helhum/dotenv-connector)
* [helhum/typo3-console](https://github.com/TYPO3-Console/TYPO3-Console)
* [sourcebroker/deployer-extended-typo3](https://github.com/sourcebroker/deployer-extended-typo3)
* [ssch/typo3-encore](https://github.com/sabbelasichon/typo3_encore)
* [typo3-contentblocks/contentblocks-reg-api](https://github.com/TYPO3-Initiatives/content-block-registration-api)
* [christophlehmann/imageoptimizer](https://github.com/christophlehmann/imageoptimizer)
* [causal/image_autoresize](https://github.com/xperseguers/t3ext-image_autoresize)
* [plan2net/webp](https://github.com/plan2net/webp)
* [lavitto/typo3-form-to-database](https://gitlab.com/lavitto/typo3-form-to-database)
* [tritum/form-element-linked-checkbox](https://github.com/tritum/form_element_linked_checkbox)
* [georgringer/news](https://github.com/georgringer/news)
* [baschte/content-animations](https://github.com/baschte/content-animations)
